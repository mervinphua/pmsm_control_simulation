## 二、核心算法：MPFMPCCController 类

### 2.1 类设计

```
MPFMPCCController
├── 初始化参数
│   ├── Ts: float          ← 控制周期 (如 1/15000 s)
│   ├── Udc: float         ← 直流母线电压
│   └── Vmax: float        ← 最大电压幅值
│
├── 内部状态
│   ├── id_prev, iq_prev   ← 上一时刻电流 i(k-1)
│   ├── ud_prev, uq_prev   ← 上一时刻电压 u(k-1)
│   ├── ud_prev2           ← 上两时刻电压 u(k-2)  [用于公式22]
│   ├── Ed_integral: float ← ∫Ed·dt 误差积分
│   ├── M: float           ← 积分增益因子
│   └── alpha: float       ← 平衡因子 α
│
├── 方法
│   ├── update()           ← 主更新函数（每个控制周期调用一次）
│   ├── _predict_current() ← 预测电流（公式8-10）
│   ├── _cost_function()   ← 代价函数（公式7）
│   └── _update_alpha()    ← 更新平衡因子（公式22, 31）
│
└── 8个开关状态的电压矢量表（固定查表）
    S0 = [0, 0, 0] → V0 = 0
    S1 = [1, 0, 0] → V1 = (2/3)Udc ∠0°
    S2 = [1, 1, 0] → V2 = (2/3)Udc ∠60°
    S3 = [0, 1, 0] → V3 = (2/3)Udc ∠120°
    S4 = [0, 1, 1] → V4 = (2/3)Udc ∠180°
    S5 = [0, 0, 1] → V5 = (2/3)Udc ∠240°
    S6 = [1, 0, 1] → V6 = (2/3)Udc ∠300°
    S7 = [1, 1, 1] → V7 = 0
```

### 2.2 update() 主流程

```
输入: id_ref, iq_ref, id(k), iq(k), ωe, θe, dt
输出: vd_opt, vq_opt (最优电压)

步骤:
┌─────────────────────────────────────────────────────┐
│  1. 读取历史值                                       │
│     id_prev = 上次保存的id                           │
│     iq_prev = 上次保存的iq                           │
│     ud_prev = 上次保存的ud                           │
│     uq_prev = 上次保存的uq                           │
│     ud_prev2 = 更早一次保存的ud                      │
├─────────────────────────────────────────────────────┤
│  2. 计算电流差和电压差                                │
│     Δid = id(k) - id_prev                            │
│     Δiq = iq(k) - iq_prev                            │
│     Δud_k = ud(k) - ud_prev      ← 当前电压差       │
│     Δuq_k = uq(k) - uq_prev                         │
├─────────────────────────────────────────────────────┤
│  3. 一步延迟补偿（公式24-25）                         │
│     先预测 i(k+1) 在上一时刻选中的开关状态作用下       │
│     Idq(k+1) = Idq(k) + ΔIdq_pred(k+1)              │
├─────────────────────────────────────────────────────┤
│  4. 遍历8个开关状态，选最优                            │
│                                                      │
│  for 每个开关状态 S in [S0, S1, ..., S7]:            │
│      │                                               │
│      ├─ 4a. 计算该状态的Udq(k+1)                      │
│      │  将S从abc坐标变换到dq坐标：                    │
│      │  [ua, ub, uc] = abc_voltage(S, Udc)           │
│      │  [uα, uβ] = clarke(ua, ub, uc)               │
│      │  [ud_k1, uq_k1] = park(uα, uβ, θe)           │
│      │                                               │
│      ├─ 4b. 计算电压差（相对于当前电压）               │
│      │  Δud_k1 = ud_k1 - ud(k)                       │
│      │  Δuq_k1 = uq_k1 - uq(k)                       │
│      │                                               │
│      ├─ 4c. 预测电流变化（公式10）                    │
│      │  Δid_pred = Δid + α·Δud_k1 + Ts·ωe·Δiq       │
│      │  Δiq_pred = Δiq + α·Δuq_k1 - Ts·ωe·Δid       │
│      │                                               │
│      ├─ 4d. 预测(k+2)时刻电流（公式8, 26）           │
│      │  id_pred = id(k+1) + Δid_pred                 │
│      │  iq_pred = iq(k+1) + Δiq_pred                 │
│      │                                               │
│      └─ 4e. 计算代价函数（公式7）                     │
│         g = (id_ref - id_pred)² + (iq_ref - iq_pred)²│
│                                                      │
│  选出 g_min 对应的最优开关状态 S_opt                   │
├─────────────────────────────────────────────────────┤
│  5. 更新平衡因子 α（公式31, 22）                      │
│     Ed = id_ref - id(k)                              │
│     Ed_integral += Ed × dt                           │
│                                                      │
│     if |Ed + Ed_prev| > ε:   ← 防除零               │
│         M = ³√(4·Δud_prev2 / (Ed + Ed_prev)²)       │
│     else:                                            │
│         M = M (保持)                                 │
│                                                      │
│     α = Ts / (0.1 + M × Ed_integral)                │
│                                                      │
│     保存 Ed_prev = Ed                                │
│     保存 Δud_prev2 = ud(k-1) - ud(k-2)              │
├─────────────────────────────────────────────────────┤
│  6. 保存历史值，供下一周期使用                         │
│     id_prev = id(k)                                  │
│     iq_prev = iq(k)                                  │
│     ud_prev = ud(k)                                  │
│     uq_prev = uq(k)                                  │
│     ud_prev2 = ud(k-1)  ← 注意：这是k-2时刻的u       │
├─────────────────────────────────────────────────────┤
│  7. 返回 S_opt 对应的 Vd, Vq                          │
└─────────────────────────────────────────────────────┘
```

### 2.3 关键公式汇总

| 公式 | 表达式 | 说明 |
|:----:|:-------|:-----|
| (8) | $I_{dq}^p(k+1) = I_{dq}(k) + \Delta I_{dq}(k+1)$ | 预测电流 = 当前电流 + 预测变化量 |
| (10) | $\Delta I_{dq}(k+1) = \Delta I_{dq}(k) + \alpha \Delta U_{dq}(k) + \beta \|\Delta I_{dq}(k)\| \Delta I_{dq}(k)^{-1}$ | **核心公式**：预测电流变化由三部分组成 |
| (7) | $g = (i_d^{ref} - i_d^p)^2 + (i_q^{ref} - i_q^p)^2$ | 代价函数 |
| (22) | $M = \sqrt[3]{\frac{4}{[E_d(k)+E_d(k-1)]^2} \cdot [u_d(k-1)-u_d(k-2)]}$ | 自适应增益因子 |
| (31) | $\alpha_k = \frac{T_s}{0.1 + M \cdot \int E_d dt}$ | 平衡因子α更新 |
| β | $\beta = \begin{bmatrix} T_s\omega_e & 0 \\ 0 & -T_s\omega_e \end{bmatrix}$ | 平衡因子β（已知） |

### 3.2 MPF-MPCC 仿真 (`simulation_mpc.py`)

```
初始化:
  motor = BLDCMotor()          ← 电机模型
  inverter = ThreePhaseInverter() ← 逆变器
  mpc = MPFMPCCController(Ts=1/15000, Udc=24) ← MPC控制器
  speed_pi = PIController(...) ← 速度环 PI (只用速度环，电流环用MPC)

主循环 (每个控制周期):
  1. 读电机状态: id, iq, ωm, θe
  2. 速度环PI: iq_ref = speed_pi.update(ω_ref - ωm)
  3. 电流环MPC: vd, vq = mpc.update(id_ref=0, iq_ref, id, iq, ωe, θe)
     └→ 内部遍历8个开关状态，选最优
  4. 逆Park变换: vα, vβ = inv_park(vd, vq, θe)
  5. SVPWM: da, db, dc = inverter.svpwm(vα, vβ)
  6. 逆变器输出: va, vb, vc = inverter.apply(da, db, dc)
  7. 施加到电机: motor.step(vd_act, vq_act, dt)
  8. 记录数据
```

### 3.3 关键区别：PI-FOC vs MPF-MPCC

| 环节 | PI-FOC | MPF-MPCC |
|:----|:-------|:----------|
| **速度环** | PI控制器 | PI控制器（相同） |
| **id电流环** | PI控制器 | MPF-MPCC算法 |
| **iq电流环** | PI控制器 | MPF-MPCC算法 |
| **需要电机参数?** | ✅ 需要R, L | ❌ **不需要** |
| **PWM生成** | SVPWM | 直接输出最优开关状态 |

## 六、伪代码（Python风格）

```python
class MPFMPCCController:
    def __init__(self, Ts, Udc, speed_pi_gains):
        self.Ts = Ts
        self.Udc = Udc
        self.Vmax = Udc / math.sqrt(3)  # 最大相电压
        
        # 速度环PI（电流环用MPC代替）
        self.speed_pi = PIController(*speed_pi_gains, -Iq_max, Iq_max)
        
        # 历史值
        self.id_prev = 0.0
        self.iq_prev = 0.0
        self.ud_prev = 0.0
        self.uq_prev = 0.0
        self.ud_prev2 = 0.0  # u(k-2) for M calculation
        
        # 平衡因子状态
        self.Ed_integral = 0.0
        self.Ed_prev = 0.0
        self.M = 0.1
        self.alpha = 10 * Ts  # 初始值 α₀ = 10·Ts
        
        # 8个开关状态的abc电压 (V)
        self.switch_states = [
            [0, 0, 0],  # S0: V0
            [1, 0, 0],  # S1: V1
            [1, 1, 0],  # S2: V2
            [0, 1, 0],  # S3: V3
            [0, 1, 1],  # S4: V4
            [0, 0, 1],  # S5: V5
            [1, 0, 1],  # S6: V6
            [1, 1, 1],  # S7: V7
        ]
    
    def switch_to_abc(self, S, Udc):
        """开关状态 → 三相电压"""
        ua = (2*S[0] - S[1] - S[2]) / 3 * Udc
        ub = (2*S[1] - S[0] - S[2]) / 3 * Udc
        uc = (2*S[2] - S[0] - S[1]) / 3 * Udc
        return ua, ub, uc
    
    def abc_to_dq(self, ua, ub, uc, theta_e):
        """三相电压 → dq电压"""
        u_alpha = (2/3) * (ua - 0.5*ub - 0.5*uc)
        u_beta = (2/3) * (math.sqrt(3)/2 * ub - math.sqrt(3)/2 * uc)
        ud = u_alpha * math.cos(theta_e) + u_beta * math.sin(theta_e)
        uq = -u_alpha * math.sin(theta_e) + u_beta * math.cos(theta_e)
        return ud, uq
    
    def update(self, omega_ref, omega_m, id_k, iq_k, ud_k, uq_k, theta_e, dt):
        """主更新函数"""
        omega_e = omega_m * self.P  # 电角速度
        
        # === Step 1: 速度环 ===
        iq_ref = self.speed_pi.update(omega_ref - omega_m, dt)
        id_ref = 0.0  # MTPA: id=0控制
        
        # === Step 2: 计算电流差和电压差 ===
        delta_id = id_k - self.id_prev
        delta_iq = iq_k - self.iq_prev
        
        # === Step 3: 一步延迟补偿 ===
        # 上一时刻选中的开关状态作用下的(k+1)电流
        # ... (公式24-25)
        
        # === Step 4: 遍历8个开关状态 ===
        g_min = float('inf')
        best_vd = 0.0
        best_vq = 0.0
        
        for S in self.switch_states:
            # 4a. 开关状态 → abc → dq电压
            ua, ub, uc = self.switch_to_abc(S, self.Udc)
            ud_k1, uq_k1 = self.abc_to_dq(ua, ub, uc, theta_e)
            
            # 4b. 电压差
            delta_ud_k1 = ud_k1 - ud_k
            delta_uq_k1 = uq_k1 - uq_k
            
            # 4c. 预测电流变化 (公式10)
            # ΔIdq_pred = ΔIdq + α·ΔUdq + β·|ΔIdq|·ΔIdq⁻¹
            # 其中 β·|ΔIdq|·ΔIdq⁻¹ = [Ts·ωe·Δiq, -Ts·ωe·Δid]ᵀ
            delta_id_pred = delta_id + self.alpha * delta_ud_k1 \
                          + self.Ts * omega_e * delta_iq
            delta_iq_pred = delta_iq + self.alpha * delta_uq_k1 \
                          - self.Ts * omega_e * delta_id
            
            # 4d. 预测(k+2)电流 (公式8, 26)
            id_pred = id_k + delta_id_pred  # 简化，实际需用延迟补偿后的值
            iq_pred = iq_k + delta_iq_pred
            
            # 4e. 代价函数 (公式7)
            g = (id_ref - id_pred)**2 + (iq_ref - iq_pred)**2
            
            if g < g_min:
                g_min = g
                best_vd = ud_k1
                best_vq = uq_k1
        
        # === Step 5: 更新平衡因子α (公式31, 22) ===
        Ed = id_ref - id_k
        self.Ed_integral += Ed * dt
        
        if abs(Ed + self.Ed_prev) > 1e-10:
            delta_ud_prev = ud_k - self.ud_prev  # ud(k) - ud(k-1)
            self.M = (4 * abs(self.ud_prev2) / ((Ed + self.Ed_prev)**2 + 1e-10)) ** (1/3)
            self.M = max(0.001, min(self.M, 1000))  # 限幅
        
        self.alpha = self.Ts / (0.1 + self.M * abs(self.Ed_integral) + 1e-10)
        self.alpha = max(self.Ts * 10, min(self.alpha, 1.0))  # 限幅
        
        # === Step 6: 保存历史值 ===
        self.ud_prev2 = ud_k - self.ud_prev  # 保存Δud(k-1)
        self.id_prev = id_k
        self.iq_prev = iq_k
        self.ud_prev = ud_k
        self.uq_prev = uq_k
        self.Ed_prev = Ed
        
        # 电压限幅
        vmag = math.hypot(best_vd, best_vq)
        if vmag > self.Vmax:
            scale = self.Vmax / vmag
            best_vd *= scale
            best_vq *= scale
        
        return best_vd, best_vq
```

## 八、注意事项

1. **平衡因子α初始化**：$\alpha_0 = 10 \cdot T_s$（论文公式30），否则电机无法启动
2. **防除零**：公式22和31的分母要加小量 `1e-10`
3. **限幅保护**：M和α都需要上下限幅，防止发散
4. **一步延迟**：论文用了(k+2)预测，代码里要对应处理好时序
5. **调试顺序**：先跑通电机空载稳态，再加阶跃和负载扰动
