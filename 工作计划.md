## 二、开源代码资源

### GitHub项目1：`nktkt/bldc-foc-simulation`

| 项目 | 内容 |
|------|------|
| **地址** | [https://github.com/nktkt/bldc-foc-simulation](https://github.com/nktkt/bldc-foc-simulation) |
| **协议** | MIT（可自由使用）|
| **语言** | Python |
| **内容** | BLDC电机FOC仿真完整框架：电机模型、SVPWM逆变器、Clarke/Park变换、级联PI控制（速度环+电流环）|
| **用途** | **作为基础框架**，将其中的PI控制器替换为MPC控制器，对比两者性能 |
| **代码结构** | ```
bldc/
├── motor.py         # BLDC电机dq轴模型
├── inverter.py      # 三相逆变器 + SVPWM
├── transforms.py    # Clarke/Park坐标变换
├── controller.py    # PI控制器 + FOC控制器
└── __init__.py
simulation.py        # 闭环仿真主程序
plots.py             # 结果可视化
``` |
